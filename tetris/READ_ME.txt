TETRIS GAME

GAME DESCRIPTION:
The game contains several game modes. Add new blocks by putting a tick at the "Additional blocks" form.
If you would like to start a new game, put a tick in front of "Forget previous scores". You may choose from 
three speeds of game to change the game`s difficulty. As well there are two themes - dark and light - available.


ATTENTION: THERE IS NO POSSIBILITY TO REMOVE THE TICKS FROM "ADDITIONAL BLOCKS" AND "FORGET PREVIOUS SCORES"
FORMS. IF YOU WANT TO GO BACK AND REMOVE A TICK, YOU MUST START THE GAME AND PRESS ESCAPE.

DEFAULT PARAMETERS:
*dark theme
*medium speed
*no additional blocks
*remember last best score

GAME INSTRUCTIONS:

UP arrow - rotate the block
DOWN arrow - accelerate the falling block
LEFT arrow - move the block to the left
RIGHT arrow - move the block to the right
SPACE - pause the game
ESCAPE back to the main menu
